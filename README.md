## 2. Visão Geral do Produto

Descrição

ShadowArchive AI é uma plataforma investigativa baseada em inteligência artificial focada na organização, catalogação, exploração e contextualização de conteúdos conspiratórios documentais, históricos e geopolíticos.

A plataforma combina:

- Base documental estilo Wikipédia;
- Arquivo investigativo estilo Pastebin;
- Busca semântica via IA;
- Assistente conversacional investigativo;
- Sistema relacional de entidades;
- Catalogação de fontes;
- Exploração narrativa documental.

O sistema NÃO valida teorias como fatos absolutos.

O objetivo é permitir:

- análise documental;
- organização investigativa;
- contextualização histórica;
- descoberta relacional;
- exploração narrativa baseada em fontes indexadas.
